한국어 가독성 및 문서 품질 가이드

## 문장 길이 및 구조
- 한 문장은 25단어 이내로 작성 (최적: 15단어 내외)
- 주어와 서술어의 거리를 가깝게 유지
- 복잡한 문장은 여러 문장으로 분할
- 능동태 문장을 수동태보다 우선 사용

## 문단 구성
- 한 문단은 3-5개 문장으로 구성
- 문단별 주제문 명확화
- 문단 간 연결어구 적절히 사용
- 긴 문단은 소제목으로 구분

## 어휘 선택
- 반복되는 단어는 다른 표현으로 변경
- 전문용어 사용 시 설명 첨부
- 모호한 표현보다 구체적 표현 선호
- 외래어보다 한국어 표현 우선

## 문체 통일성
- 하나의 문서 내에서 일관된 높임법 사용
- 격식체와 비격식체 혼용 금지
- 시제 통일 (과거/현재/미래)
- 인칭 대명사 사용 일관성 유지

## 시각적 가독성
- 적절한 문장 부호 사용 (마침표, 쉼표, 느낌표)
- 불필요한 괄호나 특수문자 최소화
- 숫자나 목록은 명확한 형식으로 표시
- 강조표현은 과도하지 않게 사용
